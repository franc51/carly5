Carly este creat de Szasz Francisc, un dezvoltator software din Brașov, România. Este o platformă creată cu scopul de a eficientiza procesul de înmatriculare al vehiculelor, eliminând birocrația și timpul pierdut.

Carly este creată cu o arhitectură de tip:
CLOUD-NATIVE SERVERLESS WEB SPA

Construită cu Angular și Angular Material pe partea de frontend, Firebase BaaS ca și serviciu backend și servicii integrate pentru plăți, autentificare, notificări și managementul fișierelor.
